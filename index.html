Firebase 設定步驟教學

1 建立專案與取得設定

  1. 前往 Firebase Console，點擊「新增專案」。
  2. 進入專案後，在總覽頁面點擊「 Web」圖示新增應用程式。
  3. 輸入名稱後註冊，複製出現的 firebaseConfig 內容 (即 apiKey 到 appId 之間)。

2 開啟資料庫 (Firestore)

  1. 在左側選單點選 Build > Firestore Database。
  2. 點擊「Create database」，選擇位置 (建議 asia-east1)。
  3. 選擇 Start in Test mode (測試模式) 快速開始。

3 開啟匿名驗證 (Auth)

  1. 在左側選單點選 Build > Authentication。
  2. 點擊「Get started」，選擇 Sign-in method 分頁。
  3. 找到 Anonymous (匿名)，點擊編輯並啟用 (Enable)。

4 如何套用設定 (寫入原始碼)

若您下載了 index.html 檔案，請使用文字編輯器開啟，搜尋關鍵字 configManager 或 defaultConfig。

// 找到這段程式碼並修改引號內容：
defaultConfig: {
  apiKey: "貼上您的 API Key",
  authDomain: "貼上您的 Auth Domain",
  projectId: "貼上您的 Project ID",
  ...
}

修改完成後儲存檔案，再次開啟網頁即可使用。

5 設定資料庫安全規則 (Rules)

請前往 Firestore Database > Rules 分頁，將內容替換為以下程式碼，並點擊 Publish。

說明：此規則允許所有人讀取(轉址)，但只有透過程式自動登入(匿名)的使用者才能寫入資料，避免惡意攻擊。
